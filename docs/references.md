# References

These resources are optional further reading and viewing. They are here to help
you build intuition around animation, rigging, and character workflows; the
coursework requirements remain in the project handouts.

## Animation And Rigging

- [Doodley](https://www.youtube.com/@doodley3d): approachable visual explainers
  on 3D animation, rigging, character controls, FK/IK, walk cycles, and motion
  principles. Good companion viewing for Part 1 and Part 2.
- [SMPL](https://smpl.is.tue.mpg.de/): official site for the Skinned
  Multi-Person Linear body model. Useful background for the body model and
  skinning ideas used in Part 2.

## Recommended Starting Points

- [3D Rigging is Beautiful, Here's How It Works!](https://www.youtube.com/watch?v=3RSwjZLClRc):
  a visual introduction to skeletons, controls, and why rigging matters before
  animation.
- [git - the simple guide](https://rogerdudler.github.io/git-guide/): a compact
  Git starter for cloning, committing, pulling, and pushing.

## Tools Used In GF5

- [Viser](https://viser.studio/main/): official documentation for the Python
  3D visualization library used by the GF5 viewer.
